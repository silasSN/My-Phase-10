Phase 10 Rules: Phase 12

The game play of this iteration I made up uses most of the same rules as normal phase 10, but there are 12 harder phases instead of 10. The only other thing that is changed in this version is that instead of dealing 10 cards deal each player 12 cards instead. Everyting else is played as normal.

  • 2 sets of 3
  • A set of 3 and a run of 4
  • A set of 4 and a run of 4
  • A run of 7
  • A run of 8
  • A run of 9
  • 2 sets of 4
  • 8 cards of one color
  • A set of 5 and a set of 2
  • A set of 5 and a set of 3
  • A set of 4 and a run of 4 in one color
  • A run of 6 in one color

    • If you want a harder gameplay look at Extreme, if you want an easier gameplay look at Hard or easy.
