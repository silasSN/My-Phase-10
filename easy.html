Phase 10 Rules: Easy

The game play of this iteration I made up useing most of the same rules as normal phase 10, but there are easier phases and a few new rules. These rules include the following:

  • #1: Skips do nothing.
  • #2: You pick up either the top card from the discard pile and discard 1, or pick up the top 2 cards from the draw pile, and discrad 2 cards.
  • #3: Wilds can be used as 2 cards when they are consective (If in a set it counts as 2, if in a run it counts as 2 only when they are consective, like 1,2, not 3,6).
  • #4: Even if you don't go out you still go the the next phase.

    • All you have to do is use the phases below instead of the normal ones, then play as normal.

  • 3 sets of 2
  • A set of 3 and a run of 3
  • A set of 3 and a run of 4
  • A run of 5
  • A run of 6
  • A run of 7
  • 4 sets of 2
  • 5 cards of one color
  • A set of 3 and a set of 2
  • 2 sets of 3

    • If you want a harder gameplay look at either Normal, Hard, Extreme, or Phase 12.
