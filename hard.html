Phase 10 Rules: Hard

The game play of this iteration I made up useing the same rules as normal phase 10, but there are harder phases. All you have to do is use the phases below instead of the normal ones, then play as normal.

  • 3 sets of 3
  • A set of 4 and a run of 4
  • A set of 4 and a run of 4 in one color
  • A run of 8
  • A run on 9
  • A run of 6 in one color
  • 2 sets of 5
  • 8 cards of one color
  • A set of 6 and a set of 2
  • Run of 7 in one color and a set of 3

    • If you want a harder gameplay look at either Extreme or Phase 12.
